## 🚀 Featured Project

### [Sistema PDI — Inventory & Order Management Platform](https://github.com/GoncaloSantos2005/PCVP/tree/main)
A production-grade platform built for the Portuguese Red Cross Ribeirão delegation, managing medical supply inventory from warehouse entry to emergency dispatch. Replaced a legacy paper and Excel system, achieving a **30% improvement in operational workflow**.

Key highlights:
- Full order lifecycle management (Draft → Approved → Shipped → Received)
- FEFO algorithm for expiration management with automated alerts
- Real-time notifications, JWT authentication, role-based access control
- PDF generation, supplier price history with trend analysis
- Being expanded to all first responders in the next version
